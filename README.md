# Web-programozás 1 - Gyakorlat Beadandó
## NB I. Labdarúgó Bajnokság

Ez a webalkalmazás a Web-programozás 1 tárgy gyakorlati beadandó feladataként készült. A projekt egy Front-controller tervezési mintára épülő, PHP, HTML, CSS és JavaScript nyelveken írt, reszponzív weboldal.

### Csoporttagok és munkamegosztás
*   **Hüse Imre:** Alaprendszer, reszponzív menü, címlap, kapcsolat űrlap kliens- és szerveroldali ellenőrzése, üzenetek listázása, útválasztó (index.php) továbbfejlesztése, statikus tartalmak.
*   **Bacsó Máté:** Képek galéria, képfeltöltés bejelentkezett felhasználóknak, adatbázis táblák tervezése és exportálása, teljes CRUD (Create, Read, Update, Delete) modul a klubok kezeléséhez, GitHub dokumentáció.

### Felhasznált technológiák
*   PHP (PDO adatbázis-kezeléssel)
*   MySQL / MariaDB
*   HTML5 & CSS3 (Flexbox reszponzív dizájn)
*   JavaScript (Kliensoldali űrlap validáció)
